⚙ Cloud-Backup (GitHub Gist)

Alle Daten werden bei jedem Speichern automatisch in ein privates GitHub Gist gesichert. So gehen keine Daten verloren — auch wenn du den Browser-Cache leerst oder ein anderes Gerät nutzt.

Nur „Gist"-Berechtigung nötig. Token wird nur lokal in diesem Browser gespeichert.
Um auf einem anderen Gerät dieselben Daten zu nutzen: Token + diese ID dort eingeben.
Einmalige Einrichtung (5 Minuten): 1. github.com aufrufen → oben rechts auf dein Profilbild klicken → „Settings"
2. Ganz unten links: „Developer settings" → „Personal access tokens" → „Tokens (classic)"
3. „Generate new token (classic)" → Name z.B. „Brunsviga" → Haken bei gist → „Generate"
4. Den angezeigten Token kopieren und hier oben einfügen → „Verbinden"
