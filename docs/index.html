Daten werden bei jedem Speichern automatisch in ein privates GitHub Gist gesichert.

Nur „gist"-Berechtigung nötig.
Beim ersten Verbinden automatisch ausgefüllt.
Einmalige Einrichtung: github.com → Profilbild → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token → Haken bei gist → Generate → Token kopieren → hier einfügen

GitHub speichert automatisch jede Version. Hier kannst du frühere Stände ansehen und wiederherstellen.
